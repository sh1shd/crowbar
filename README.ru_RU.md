# Crowbar

Ответвление проекта 3X-UI для личного использования.

## Описание 3X-UI

**3X-UI** — продвинутая веб-панель управления с открытым исходным кодом, предназначенная для управления сервером Xray-core. Она предлагает удобный интерфейс для настройки и мониторинга различных протоколов VPN и прокси.
Являясь усовершенствованным ответвлением оригинального проекта X-UI, 3X-UI обеспечивает улучшенную стабильность, более широкую поддержку протоколов и дополнительные функции.
